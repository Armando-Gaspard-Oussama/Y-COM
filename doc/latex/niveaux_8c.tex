\doxysubsection{Detailed Description}
Fonctions pour la gestion de niveaux à partir de leurs fichiers. 

Les niveaux sont représentés dans leurs fichiers media/niveaux/\+XX.\+niv où XX est le numéro du niveau. Chaque fichier est une suite de caractères de chiffres sans espaces, représentant la matrice de gauche à droite et de haut en bas. Chaque caractère représente un bloc spécifique. \begin{DoxySeeAlso}{See also}
